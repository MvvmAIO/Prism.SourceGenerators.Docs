# Prism.SourceGenerators.Docs

Blazor WebAssembly documentation for **[MvvmAIO.Prism.SourceGenerators](https://github.com/MvvmAIO/Prism.SourceGenerators)**.

## Live site

**https://mvvmaio.github.io/Prism.SourceGenerators.Docs/**

Use **Settings -> Pages** with **GitHub Actions** as the publishing source if the site is not live yet.

## Languages

Strings in `wwwroot/locales/` (**en**, **zh-CN**, **ja**). Language choice is stored in `localStorage`.

## Local run

```bash
dotnet run --project Prism.SourceGenerators.Docs.csproj
```
