# 关于本站

本站是 **MvvmAIO.Prism.SourceGenerators** 的**权威、可版本化**的产品手册，源码在 **[Prism.SourceGenerators.Docs](https://github.com/MvvmAIO/Prism.SourceGenerators.Docs)**，通过 GitHub Pages 发布。

## 与其他信息源的关系

| 渠道 | 定位 |
|------|------|
| **本站** | 结构化目录、生成器语义、诊断、架构、构建与贡献说明；**深度内容以本站为准**。 |
| **[GitHub README](https://github.com/MvvmAIO/Prism.SourceGenerators/blob/master/README.md)** | 概览与快速粘贴示例；可能与文档 PR 有短暂时差。 |
| **[DeepWiki](https://deepwiki.com/MvvmAIO/Prism.SourceGenerators)** | 仓库的 AI 索引镜像，便于浏览，**不作为 API 或诊断文案的合同**。 |
| **示例仓库** | 可运行应用；见 [Samples](samples.md)。 |

编译器输出的**合同**仍以产品仓库中的 **`DiagnosticDescriptors.cs`** 为准；人读索引见 [诊断参考](diagnostics/reference.md)。

## 语言

**简体中文**与本站英文结构对齐（架构、生成器、诊断、构建等）。若某段尚未翻译，可使用 **[English](../)** 侧栏或站内搜索。

## 参与文档

文档变更走 **Prism.SourceGenerators.Docs**（Markdown、`mkdocs.yml`、可选 `styles/`）。生成器行为见 **Prism.SourceGenerators**；详见 [贡献指南](contributing.md)。
