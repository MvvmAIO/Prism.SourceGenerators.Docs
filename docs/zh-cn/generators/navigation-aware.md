# NavigationAware

在引用 **Prism.Core** 的 **partial** 类上使用 **`[NavigationAware]`**，可生成 **`Prism.Navigation.Regions.INavigationAware`** 成员。

用法与英文文档相同，见 [NavigationAware（English）](/generators/navigation-aware)。

诊断：**PSG0007**（类须为 `partial`）。
