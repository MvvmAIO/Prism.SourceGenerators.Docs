# コントリビュート

## 製品（ジェネレータ、Core、テスト）

**Prism.SourceGenerators** の **[CONTRIBUTING.md](https://github.com/MvvmAIO/Prism.SourceGenerators/blob/master/CONTRIBUTING.md)** に従ってください（Issue、ブランチ、テスト、スタイル）。

## ドキュメント（本サイト）

- リポジトリ：**[Prism.SourceGenerators.Docs](https://github.com/MvvmAIO/Prism.SourceGenerators.Docs)**  
- コンテンツ：**`docs/`** の Markdown、**`.vitepress/config.mts`**、任意の **`.vitepress/theme/custom.css`**。  
- ローカルビルド：そのリポジトリで **`dotnet run`**（README 参照）。

**診断 ID やユーザー向けメッセージ** を変えたら、製品 PR と同じか直後に **[診断リファレンス](/ja/diagnostics/reference)** を更新し、本サイトとコンパイラ出力を一致させてください。
